This SR is for deploy vue-based project and provide a frontend URL.

You need to provide a vue project repo URL and the commit ID. If the repo is private, please make sure you have the access of that repo and generate an access token. Then create a .netrc file with the access token.

For more info about how to use the .netrc, please refer https://www.erp5.com/slapos-HowTo.Use.Private.Software.Release.Gitlab
