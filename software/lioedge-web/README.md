This SR is for deploy lioedge-web and provide a frontend URL.

The lioedge-web is a private repo. Before deploy this SR, please make sure you have the access of that repo and generate an access token. Then create a .netrc file with the access token.

For more info about how to use the .netrc, please refer https://www.erp5.com/slapos-HowTo.Use.Private.Software.Release.Gitlab
